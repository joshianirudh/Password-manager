->axios
>react-datepicker
------------------------------------------------------
Database:
MongoDB-Atlas
-------------------------------------------------------
To make the project work:
->go to the password-manager folder
->open cmd and type "npm install"
->after installation is done go to the backend folder in the same directory
->open cmd and type "npm install"
->for any of the above installations, if there are any vulnerabilities, use "npm audit fix"
->if there still vulnerabilities, use "npm audit fix --force"
->it is also recomended to install nodemon globally using "npm install nodemon -g"
->while in the backend directory, check if the server is working using "nodemon server"
->to start project, go back to password manager folder and use "npm start"
->ALL THE BEST!!
